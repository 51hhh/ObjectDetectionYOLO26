# 训练环境搭建与训练步骤

## 1. 目标说明

本文档用于规范本项目的本地训练流程，包含：
- Python 环境搭建
- YOLO26 依赖安装
- 数据集准备
- 模型训练
- 训练结果查看

项目任务：单类别排球检测，类别名为 `volleyball`。

---

## 2. 环境要求

推荐环境：
- 操作系统：Windows 11
- Python：3.10 ~ 3.12
- GPU：NVIDIA 显卡
- CUDA：建议驱动支持 CUDA 12.x

本项目已验证环境：
- Python 3.12.7
- PyTorch 2.5.1 + CUDA
- Ultralytics 8.4.24
- GPU：RTX 4070 SUPER

---

## 3. 创建训练环境

在项目根目录执行：

```bash
python -m venv .venv --system-site-packages
"./.venv/Scripts/python.exe" -m pip install --upgrade pip
"./.venv/Scripts/python.exe" -m pip install ultralytics==8.4.24 pyyaml pillow
```

验证安装：

```bash
"./.venv/Scripts/python.exe" -c "import ultralytics; print(ultralytics.__version__)"
```

---

## 4. 数据文件说明

原始数据文件：
- `images.zip`：正样本压缩包
- `negative_samples.zip`：负样本压缩包

当前数据特征：
- 正样本包含 `webp + xml/json 标注`
- 负样本只包含图片，不包含标注
- 类别只有一个：`volleyball`

---

## 5. 数据集转换

执行以下命令，将原始压缩包转换为训练用数据集：

```bash
"./.venv/Scripts/python.exe" "./scripts/prepare_dataset.py"
```

转换输出目录：
- `coco/`

生成内容包括：
- `coco/images/train`
- `coco/images/val`
- `coco/images/test`
- `coco/labels/train`
- `coco/labels/val`
- `coco/labels/test`
- `coco/annotations/instances_train.json`
- `coco/annotations/instances_val.json`
- `coco/annotations/instances_test.json`
- `coco/data.yaml`
- `coco/summary.json`

说明：
- 当前脚本会自动把负样本并入训练集
- 负样本以“空标注背景图”的形式参与训练

---

## 6. 训练配置文件

训练配置文件位置：
- `configs/train_volleyball.yaml`

当前推荐配置：

```yaml
model: yolo26n.pt
data: coco/data.yaml
epochs: 120
imgsz: 640
batch: 0.8
device: 0
workers: 8
optimizer: auto
patience: 40
pretrained: true
amp: true
cache: ram
project: runs
name: volleyball_train
exist_ok: true
save_period: 10
verbose: true
```

参数说明：
- `model`：预训练模型权重
- `data`：数据集配置文件
- `epochs`：训练轮数
- `imgsz`：输入分辨率
- `batch: 0.8`：自动使用约 80% 显存计算 batch size
- `workers`：dataloader worker 数量
- `cache: ram`：将数据缓存到内存，提高吞吐

---

## 7. 启动训练

执行：

```bash
"./.venv/Scripts/python.exe" "./scripts/train_volleyball.py"
```

训练输出目录：
- `runs/volleyball_train/`

训练过程中重点关注：
- `GPU_mem`
- `box_loss`
- `cls_loss`
- `mAP50`
- `mAP50-95`

---

## 8. 训练结果文件说明

训练完成后，核心文件位于：
- `runs/volleyball_train/weights/best.pt`

## 10. 常见问题

### 10.1 GPU 利用率不高
可尝试：
- 提高 `workers`
- 使用 `cache: ram`
- 提高 `batch` 比例，如 `0.9`
- 提高 `imgsz` 到 `768`

### 10.2 误检较多
可尝试：
- 增加硬负样本
- 提高推理置信度阈值 `conf`
- 补充复杂背景数据

### 10.3 漏检较多
可尝试：
- 增加小目标、远距离样本
- 提高输入尺寸 `imgsz`
- 增强遮挡、模糊场景数据
