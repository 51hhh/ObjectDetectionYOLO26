# 部署推理文档

## 1. 目标说明

本文档用于说明如何使用训练好的 `best.pt` 模型进行：
- 单图推理
- 视频推理
- 文件夹批量推理
- 摄像头实时推理

模型文件默认位置：
- `runs/volleyball_train/weights/best.pt`

---

## 2. 推理脚本位置

推理脚本：
- `scripts/infer_volleyball.py`

脚本默认加载：
- `runs/volleyball_train/weights/best.pt`

---

## 3. 基本命令

### 3.1 单张图片推理

```bash
"./.venv/Scripts/python.exe" "./scripts/infer_volleyball.py" "你的图片路径.jpg"
```

### 3.2 视频推理

```bash
"./.venv/Scripts/python.exe" "./scripts/infer_volleyball.py" "你的视频路径.mp4"
```

### 3.3 文件夹批量推理

```bash
"./.venv/Scripts/python.exe" "./scripts/infer_volleyball.py" "你的图片文件夹"
```

### 3.4 摄像头实时推理

```bash
"./.venv/Scripts/python.exe" "./scripts/infer_volleyball.py" 0 --show
```

---

## 4. 常用参数

```bash
--weights   模型权重路径
--imgsz     推理尺寸，默认 640
--conf      置信度阈值，默认 0.25
--iou       NMS IoU 阈值，默认 0.45
--device    推理设备，如 0 或 cpu
--name      输出目录名
--save-txt  保存 txt 格式结果
--save-conf 保存置信度到 txt
--show      打开可视化窗口
```

示例：

```bash
"./.venv/Scripts/python.exe" "./scripts/infer_volleyball.py" "test.mp4" --conf 0.4 --iou 0.5 --name test_run
```

---

## 5. 输出结果位置

推理输出会保存到：
- `runs/<name>/`

例如：
- `runs/volleyball_predict/`
- `runs/test_run/`

输出内容通常包括：
- 可视化结果图/视频
- 可选 txt 检测结果

---

## 6. 生产部署建议

### 6.1 推荐使用 best.pt
生产部署建议优先使用：
- `runs/volleyball_train/weights/best.pt`

### 6.2 推理阈值建议
建议从以下参数开始：
- `conf=0.35 ~ 0.5`
- `iou=0.45`

如果误检偏多：
- 提高 `conf`

如果漏检偏多：
- 降低 `conf`
- 提高 `imgsz`

### 6.3 输入尺寸建议
- 普通场景：`imgsz=640`
- 小目标或远距离目标：`imgsz=768` 或更高

---

## 7. 部署交付建议

最小部署交付内容建议保留：

```text
configs/
scripts/infer_volleyball.py
runs/volleyball_train/weights/best.pt
.venv/            # 可选，不建议直接交付
```

更推荐的部署交付方式：
- 交付 `best.pt`
- 交付 `infer_volleyball.py`
- 提供 requirements 或环境说明

---

## 8. 推理效果验证建议

部署前建议验证以下场景：
- 单球近景
- 单球远景
- 复杂背景
- 空背景
- 多目标干扰场景
- 视频连续帧稳定性

重点观察：
- 是否误检背景
- 是否漏检小球
- 边缘位置是否稳定
- 视频中检测框是否抖动明显
